# Introduction to helpr

The `helpr` package contains numerous helpers, wrappers, and utilities
used throughout the my analysis suite. It intentionally favors base R
over higher level *tidyverse* to minimize imports. The goal is to
provide an alternative gain in functionality without the cost of
additional imports/dependencies.

------------------------------------------------------------------------

## Useful functions in `helpr`

- File-system: See
  [`?filesystem`](https://stufield.github.io/helpr/dev/reference/filesystem.md)
- Signal UI: See
  [`?signal`](https://stufield.github.io/helpr/dev/reference/signal.md)
- Symbols: See `?sybml`
- Strings:
  - [`squish()`](https://stufield.github.io/helpr/dev/reference/strings.md)
  - [`trim()`](https://stufield.github.io/helpr/dev/reference/strings.md)
  - [`pad()`](https://stufield.github.io/helpr/dev/reference/strings.md)
  - [`capture()`](https://stufield.github.io/helpr/dev/reference/strings.md)
- Handlers:
  - [`be_safe()`](https://stufield.github.io/helpr/dev/reference/handlers.md)
  - [`be_quiet()`](https://stufield.github.io/helpr/dev/reference/handlers.md)
  - [`be_hard()`](https://stufield.github.io/helpr/dev/reference/handlers.md)
- [`liter()`](https://stufield.github.io/helpr/dev/reference/liter.md):
  - List iteration, similar to
    [`purrr::imap()`](https://purrr.tidyverse.org/reference/imap.html)
    but simpler
- Calculations:
  - [`calc_ccc()`](https://stufield.github.io/helpr/dev/reference/calc_ccc.md):
    Calculate Lin’s Concordance Correlation Coefficient for two vectors
    of numeric data.
  - [`calc_ss()`](https://stufield.github.io/helpr/dev/reference/calc_ss.md):
    Calculate the sum of squared errors:
    $\sum\left( x - \bar{x} \right)^{2}$ for numeric data.
- Logic Tests:
  - [`rep_lgl()`](https://stufield.github.io/helpr/dev/reference/rep_lgl.md):
    Are all the elements of a vector identical?
  - [`is_monotonic()`](https://stufield.github.io/helpr/dev/reference/logicals.md):
    Are the numerical elements of a vector monotonically increasing or
    decreasing?
  - [`is_logspace()`](https://stufield.github.io/helpr/dev/reference/logicals.md):
    Does a given object (`vector`, `data.frame`, `tbl_df`) appear to be
    in log-space? **Note:** this function is biased to proteomic data
    and should not be expected to be accurate for other applications
    out-of-the-box.
- [`cross_tab()`](https://stufield.github.io/helpr/dev/reference/cross_tab.md)
  - Create a contingency table of counts generated by cross-classifying.
- [`diff_vecs()`](https://stufield.github.io/helpr/dev/reference/diff_vecs.md)
  - Generate all diffs of two vectors of (typically character) data.
    Returns both sided `setdiff`, `union` and `intersect`.
- [`dater()`](https://stufield.github.io/helpr/dev/reference/dater.md)
  - Generate a standardized data in (by default) `YYYY-MM-DD`.

## Examples

### `cross_tab()`

You must not “quote” the passed arguments, unquoted strings are fine and
are parsed by NSE (non-standard evaluation). The `...` can take on
either one or two column names, any additional parameters are passed to
\[table()\]:

``` r
# 1 factor
cross_tab(mtcars, cyl)        # 1 factor
#> cyl
#>   4   6   8 Sum 
#>  11   7  14  32

cross_tab(mtcars, cyl, gear)  # 2 factors
#>      gear
#> cyl    3  4  5 Sum
#>   4    1  8  2  11
#>   6    2  4  1   7
#>   8   12  0  2  14
#>   Sum 15 12  5  32

cross_tab(mtcars, cyl, gear, useNA = "always")  # NAs
#>       gear
#> cyl     3  4  5 <NA> Sum
#>   4     1  8  2    0  11
#>   6     2  4  1    0   7
#>   8    12  0  2    0  14
#>   <NA>  0  0  0    0   0
#>   Sum  15 12  5    0  32

cross_tab(mtcars, cyl, gear, dnn = list("group1", "group2"))  # names
#>       group2
#> group1  3  4  5 Sum
#>    4    1  8  2  11
#>    6    2  4  1   7
#>    8   12  0  2  14
#>    Sum 15 12  5  32
```

### `calc_ccc()`

Calculate the `ccc` for two numeric vectors (visualize by concordance):

``` r
x <- rnorm(100, mean = 10, sd = 0.5)
y <- x + rnorm(100, sd = 0.1)   # add random scatter
ccc <- calc_ccc(x, y)
plot(x, y, pch = 21, cex = 1.75, col = NA,
     bg = rgb(red = 0, green = 0, blue = 0, alpha = 0.5),   # black w alpha
     main = sprintf("The CCC = %0.3f with significance of p = %0.3f",
                    ccc$rho.c, ccc$p.value)
)
abline(0, 1, lty = 2, col = "blue", lwd = 1.5)
```

### `calc_ss()`

Calculate the sum of squared differences for a numeric vector. Used
ubiquotously in generating variances and standard deviations within
other contexts (e.g. ANOVA, CVs):

``` r
x <- rnorm(100, mean = 10, sd = 0.5)
calc_ss(x)
#> [1] 18.85493

calc_ss(x) / (length(x) - 1)    # variance
#> [1] 0.1904538

all.equal(var(x), calc_ss(x) / (length(x) - 1))    # TRUE
#> [1] TRUE
```

### `rep_lgl()`

``` r
rep_lgl(letters)
#> [1] FALSE

rep_lgl(rep("A", 250))
#> [1] TRUE

rep_lgl(c("B", rep("A", 250)))
#> [1] FALSE
```

### `is_monotonic()`

``` r
is_monotonic(1:100)
#> [1] TRUE

is_monotonic(seq(-100, 100, by = 5))     # up
#> [1] TRUE

is_monotonic(seq(100, -100, by = -5))    # down
#> [1] TRUE

is_monotonic(rnorm(10))
#> [1] FALSE
```

### `is_logspace()`

``` r
# A numeric vector
x <- rnorm(30, mean = 1000)
is_logspace(x)
#> [1] FALSE

is_logspace(log(x))
#> [1] TRUE
```

``` r
is_logspace(data) # FALSE

# log10-transform
for ( i in grep("^ft", names(data)) ) {
  data[[i]] <- log10(data[[i]])
}

is_logspace(data) # base 10; TRUE
```
